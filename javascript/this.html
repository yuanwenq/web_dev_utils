    this对象的理解

    # 定义

    Javascript中的this关键字表示当前函数执行的上下文对象，它指向一个对象(运行时自动生成的一个内部对象)，这个对象通常是函数调用时所在的对象(称之为函数上下文)。

    this可以在函数内部使用，并且它的指向是根据函数的调用方式动态确定的，因此在不同的上下文，this所指向的对象也是不同的。

    this在函数执行过程中，一旦确定则无法更改

    # this 指向的几种情况

    1. 全局上下文的this指向全局对象
      • 浏览器为window对象
      • Node.js中为global对象
    2. 函数作为对象的方法被调用时，函数上下文的this指向该对象
    var obj = {
  a: 1,
  f: function() {
    console.log(this.a);
  }
}

obj.f(); // 输出 1

    1. 通过call、apply、bind等方法来改变函数执行时的this指向
    var obj1 = { name: 'obj1' };
var obj2 = { name: 'obj2' };

function sayHello() {
  console.log('Hello, ' + this.name);
}

// this 为 window
sayHello(); // 输出 Hello,

// 使用 call 方法，传递参数列表
sayHello.call(obj1, 'param1', 'param2'); // 输出：Hello, obj1

// 使用 apply 方法，传递参数数组
sayHello.apply(obj2, ['param1', 'param2']); // 输出：Hello, obj2

// 使用 bind 方法，返回新的函数
var newFunc = sayHello.bind(obj1);
newFunc(); // 输出：Hello, obj1

    1. 在构造函数中，this 指向当前创建的实例对象。

    在构造函数中，使用this关键字来引用当前创建的对象实例。当使用new操作符来调用构造函数时，Javascript会自动创建一个新的空白对象，并将该对象赋值给this关键字。在构造函数中，可以通过this来添加属性和方法，并在创建对象实例时访问它们。

    function Person(name, age) {
  this.name = name;
  this.age = age;
  this.speak = function() {
    console.log(`My name is ${this.name}, and I am ${this.age} years old.`);
  }
}

var john = new Person('John', 30);
john.speak(); // 输出：My name is John, and I am 30 years old.

    1. 在箭头函数中，this 指向上一层的执行上下文，而不是函数执行时所在的上下文。

    例子一

    const obj = {
  num: 100,
  myFunc: function () {
    console.log(this.num); // 输出：100
    const myArrowFunc = () => {
      console.log(this.num); // 输出：100
    };
    myArrowFunc();
  }
};
obj.myFunc();

    例子二

    const obj = {
  num: 100,
  myFunc: () => {
    console.log(this.num); // undefined
  }
};
obj.myFunc();

    # 优先级

    new绑定优先级 > 显示绑定优先级 > 隐式绑定优先级 > 默认绑定优先级

    # 隐式绑定 VS 显示绑定

    function foo() {
    console.log( this.a );
}

var obj1 = {
    a: 2,
    foo: foo
};

var obj2 = {
    a: 3,
    foo: foo
};

obj1.foo(); // 2
obj2.foo(); // 3

obj1.foo.call( obj2 ); // 3
obj2.foo.call( obj1 ); // 2
